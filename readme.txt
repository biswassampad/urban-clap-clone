

“Service booking app / Service booking Product” is android app based one stop solution to provide various type of services for repair, installation and solution. 

________About the software__________ 

This software is totally written in native android, codeignitor , mysql and php. 

____Features____ 

Admin Panel: 
Dashboard: Assigned, Pending Completed, App Users, Total Paid Amount.
Booking: Assigned, Pending, Started, Completed, Canceled & Job Card manage.
Categories: Add, delete and update services.
Service man: Add, remove and update technician/Engineer in team.
App User Manage: Client details management.
Schedule: Change service time / leave / off day.
Offer: Add, delete, update new offer.
Send Notification to user from web.
Page management: About / Contact / Additional pages.
Site Management: currency / email / notification setting.

Service booking App (User Side): 
Book Service: Select, book and cancel service.
Service progress: Check progress status of booked service.
Team: Get details of service man contact / Working hour / Rating.
Contact us: Admin contact details.

Service Man app (Technician / Engineer Side):
Profile: Update profile password/ contact details.
Assigned: Check assigned Service details.
Completed: Check completed service Details.
