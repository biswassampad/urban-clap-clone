App Features

 

Features :

  • Easy and cool design
  • Dashboard: Assigned, Pending Completed, App Users, Total Paid Amount.
  • Booking: Assigned, Pending, Started, Completed, Canceled & Job Card manage.
  • Categories: Add, delete and update services.
  • Service man: Add, remove and update technician/Engineer in team.
  • App User Manage: Client details management.
  • Schedule: Change service time / leave / off day.
  • Offer: Add, delete, update new offer.
  • Send Notification to user from web.
  • Page management: About / Contact / Additional pages.
  • Site Management: currency / email / notification setting.

Package Files

Download latest source code from codecanyon. Package file include following list of contents. 

1. Backend [Folder : source of backend site]

2. Android App [Folder : source of complete app]

3. Document [Folder include help file how to install and run app]

4. Sample Database [Folder include sample database file]

Base requirment

  • This application is developed in Android Native code. For running android application you must have installed Android Studio. Download and Install android studio from following link. https://developer.android.com/studio
  • Backend is developed in  CodeIgnitor 3.x version. For this you must required Linux hosting.  find help regarding codeigniter  :  https://codeigniter.com/docs     
  • For database required MySql.

Installation of Application

  • Create database
  • Copy downloaded zip and extract in website main directory
  • Update database, Usename and Password in following file
  • “\application\config\database.php”

  • Open Website
  • Following Screen will display – Fill all details as per required.

  • After filling all details click on “Add” button.
  • Script auto config website as per details. ( If any error checked filled details )
  • After auto config site auto redirect to login screen as per below.

  • Insert username & password which you define in previous screen click on “sign in” for login.

  • Your site is ready now you can use it as per requirement.
