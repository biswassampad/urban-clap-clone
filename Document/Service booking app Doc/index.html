App Features

Service Booking app is for to book appointment for various type of services like Plambing, Cleaning, Painting, Home appliances, Lighting and Decoration, Car Service, Bike services, Electric Work, Carpentry Services, Mason Work, Spa Services, Gardning, Security Services, Taxi Services, Printer repair, Packers And Movers, Water proffing, Pop, PC/Laptop repair, CCTV Camera, Pest Control, AC Service and repair, Refrigerator Repair, Water Purifier / Ro Repair, Mobile / Tablet Repair, TV / LCD / LED / Plazma Repair, Air Cooler Repair, Geyser Water , Heater Repair, Miacrowave Repair, Washing Machine Repair, Solar repair, Many More other services.

Features :

1. Easy and cool design

2. Simple user registration and Login with app

3. Simple step to book service

                a. Choose service

                b. Choose which fault will repair and book

                c. Go to cart and view offers or apply it otherwise click to continue

                d. Choose date and time. and choose address

                e. Book service

4. Complete information of Service Booking app

                a. Service price, review, service persion list

                b. View offers

                c. Change and add Delivery address

                e. Add reviews on completed services

5. Get Contact

6. User Account  (Change profile,  My Services, Change password, Change address)

7. App allow to use multi language if you need.

Base requirment

  • This application is developed in Android Native code. For running android application you must have installed Android Studio. Download and Install android studio from following link. https://developer.android.com/studio
  • Backend is developed in  CodeIgnitor 3.x version. For this you must required Linux hosting.  find help regarding codeigniter  :  https://codeigniter.com/docs    
  • For database required MySql.

Installation of Application

Android Application source is given in  "Application" folder.

                - Open Android Studio

 

Step 1  :   Choose "Open Existing Android Studio Project"

Step 2 : Choose project folder from your disk and direct open it. 

If android studio ask you for missing plugins and components then install all of it. Wait until complete building gradle process done.

This will take some time to create gradle process.    

 

 

Configure Android App :

 - Configure your base url to connect with your backend in your application open following files.

1.  "app/src/main/java/Config/BaseURL.java"

2.  Update base url with your  "Backend" url installed.

This will syncronize data to your application.

Create AVD and run app. 

How to create AVD in follow this link. https://developer.android.com/studio/run/managing-avds.html

 

Change APP ID / Bundle Name : 

1. in app level folder  edit build.bundle file and update here application id  this will change your application name, 

"app/build.bundle"  change "applicationId"

3. update "google-service.json" file with new json

 

 

Change App Language and Strings : 

This file will change all display strings in application. 

"app/main/res/values/strings.xml"  this file contains default language of application. All defaults keywords used in application is here in this file. 

 

If you wont to add new Language in this application create  "values-ar"  folder with your  language POSTFIX this will add language to your app.  copy and past "strings.xml" file in this folder and replace all keywords values to your language.

 

 

Change App Color: 

This file will change all display colors in application. 

"app/main/res/values/color.xml"  this file contains default color of application. All defaults color used in application is here in this file.

 

 

Change App Logo: 

This file will change app logo in application. 

"app/main/res/drawable" right click on it

1) new -> Image Asset

2) browes app logo and click "next" then "ok" done
